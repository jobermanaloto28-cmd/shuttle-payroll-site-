Shuttle Payroll Automation

An internal payroll processing tool built for a shuttle transport service in Calamba, Laguna, Philippines. This application automates driver salary computation and payslip generation.

How it works

Drivers submit trip data via Telegram. The system extracts the relevant details, computes payroll using fixed company rate tables stored in Google Sheets, and generates an official payslip document using a Google Docs template, delivered back to the driver as a PDF.

This is an internal tool used only by the shuttle service's own administrators and drivers. It is not a public product or service.
